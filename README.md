# downdetector
Detects if a site is down

## Setup
1. Create a folder
```shell
cd /var/www
mkdir downdetect
```
2. Clone the repository
```shell
git clone https://github.com/NerdOfLinux/downdetector.git .
```
3. Configure your PHP-enabled web server
4. Install `php-curl`:
```shell
sudo apt install php-curl
```
5. Test it out!
